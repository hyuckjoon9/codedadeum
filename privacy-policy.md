# 개인정보처리방침

**코드다듬기** Chrome 확장 프로그램은 사용자의 개인정보를 수집하지 않습니다.

## 수집하는 데이터

본 확장 프로그램은 어떠한 개인정보도 수집하지 않습니다.

## 저장하는 데이터

사용자가 팝업에서 선택한 변환 옵션(클래스명 변환, 주석 제거 등)은
`chrome.storage.sync`를 통해 사용자의 Google 계정에 저장됩니다.
이 데이터는 외부 서버로 전송되지 않으며, 개발자는 접근할 수 없습니다.

## 제3자 데이터 공유

수집된 데이터가 없으므로 제3자와 공유하는 데이터도 없습니다.
